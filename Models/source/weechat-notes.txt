See all weechat options:
/set weechat.*

See all IRC plugin options:
/set irc.*

Help on an option:

/help weechat.option.name

/connect network_name

By default, server buffers and core buffers are merged.
Ctrl+x will switch between them.

Can permanately disable with:
/set irc.look.server_buffer indepenent

Send a "private" message:
/query nick Message Goes Here.
(why not /msg ?)

/buffer and /window:

Split windows vertically at 33%
/window splitv 33

Un-split a window:
/window merge (



Add an IRC server:
/server add name server.address.com

Server options:

/set irc.server.name.nicks "mynick,mynick1,mynick2"
/set irc.server.name.username "myusername"
/set irc.server.name.realname "Real Name"
/set irc.server.name.autoconnect on
/set irc.server.name.autojoin "#channel1,#channel2"
/set irc.server.name.option null (unsets an option)

SSL:
/set irc.server.name.addresses "server.address.com/9999"
/set irc.server.name.ssl on
/set irc.server.example.ssl_verify off

SASL
/set irc.server.name.sasl_username "mynick"
/set irc.server.name.sasl_password "xxxxxxx"

Runs command after connecting to server:
/set irc.server.name.command "/msg nickserv identify xxxxxxx"


Enable mouse at startup:
/set weechat.look.mouse on

alt+m or /mouse enable to toggle mouse functionality.

Can secure the sec.conf file with a password:
/secure passphrase password_goes_here

Requries password be entered on startup of weechat.
Otherwise sec.conf is plaintext.

Data can be added to sec.conf by:
/secure set tag data
/secure set freenode mypassword

Secure data can be displayed in a dedicated buffer by:
/secure

Private data can be passed to some options (like passwords).
Use the format: ${sec.data.tag}
where tag is is the name of the secure data.
Ex: /set irc.server.freenode.sasl_password "${sec.data.freenode}"

see encryption options:
/set sec.crypt.*
